# Programming Puzzle Game

A simple game to teach basic programming concepts.

[Play the game](/assets/2026-08-02-program_game/game.html)


## About This Game

This game teaches basic programming concepts by having you give commands to move a character through a 7x7 grid.

## How to Play

1. Use the code blocks to create a program
2. Click "Run" to execute it
3. Try to reach the green goal square

## Learning Outcomes

- Sequential thinking
- Basic command structures
- Problem solving

---

*The game is built with vanilla JavaScript and runs entirely in your browser.*
