Programming Puzzle Game

Teach programming concepts through interactive puzzles

How to Play

  • Click code blocks to build your program
  • Program executes commands in order
  • Use "Run Program" to execute all at once
  • Use "Step" to execute one command at a time
  • Collect all coins before reaching the green flag to win!

This game runs entirely in your browser. No data is saved or sent to any server.
