Programming Puzzle Game

Teach programming concepts through interactive puzzles

How to Play

  1. Click code blocks to build your program
  2. Program executes commands in order
  3. Use "Run Program" to execute all at once
  4. Use "Step" to execute one command at a time
  5. Reach the green flag to win!

This game runs entirely in your browser. No data is saved or sent to any server.
